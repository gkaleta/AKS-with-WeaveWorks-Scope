# Up and running with WeaveWorks with AKS
## Prerequsites
1. Running AKS cluster 
2. Full admin access to AKS cluster
3. AZ CLI installed
4. Kubectl installed

## Ensure you have access to your cluster

Run `kubectl get nodes`
